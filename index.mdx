# Senal Market Maker API

Senal 做市商 API 为机构交易者提供低延迟、高吞吐的预测市场做市接口。API 设计对齐 [Polymarket CLOB API](https://docs.polymarket.com)，方便已有做市系统快速接入。

## Key Features

<CardGroup cols={2}>
  <Card title="Dual-Layer Auth" icon="shield-halved">
    L1 EIP-712 钱包签名 + L2 HMAC-SHA256 请求签名，兼顾安全与性能
  </Card>
  <Card title="Order Types" icon="layer-group">
    支持 GTC、GTD、FOK、FAK 四种订单类型，以及 Post-Only 模式
  </Card>
  <Card title="Batch Operations" icon="boxes-stacked">
    原子批量下单（最多 15 笔）、批量撤单（最多 3000 笔）
  </Card>
  <Card title="Real-time WebSocket" icon="bolt">
    订单簿、成交、订单状态、持仓变更实时推送
  </Card>
</CardGroup>

## Base URL

```
https://api.senal.market/v1
```

## Rate Limits

| Endpoint | Limit |
|----------|-------|
| REST API | 100 req/s per API Key |
| WebSocket | 5 connections per API Key |
