---
title: "查询订单详情"
api: "GET /v1/mm/orders/{id}"
description: "查询单笔订单详情"
---

## 请求头（L2 认证）

<ParamField header="SENAL-API-KEY" type="string" required>API Key</ParamField>
<ParamField header="SENAL-SIGNATURE" type="string" required>HMAC-SHA256 签名（hex）</ParamField>
<ParamField header="SENAL-TIMESTAMP" type="string" required>当前时间戳（毫秒）</ParamField>
<ParamField header="SENAL-NONCE" type="string" required>随机字符串（至少 16 字节）</ParamField>
<ParamField header="SENAL-PASSPHRASE" type="string" required>API Key 对应的 Passphrase</ParamField>

## 路径参数

<ParamField path="id" type="string" required>
  订单 ID
</ParamField>

## 响应

<ResponseField name="id" type="string">订单 ID</ResponseField>
<ResponseField name="marketId" type="string">市场 ID</ResponseField>
<ResponseField name="tokenId" type="string">Token ID</ResponseField>
<ResponseField name="side" type="string">`BUY` 或 `SELL`</ResponseField>
<ResponseField name="orderType" type="string">订单类型：`GTC`、`GTD`、`FOK`、`FAK`</ResponseField>
<ResponseField name="price" type="string">限价</ResponseField>
<ResponseField name="size" type="string">原始数量</ResponseField>
<ResponseField name="filledSize" type="string">已成交数量</ResponseField>
<ResponseField name="remainingSize" type="string">剩余数量</ResponseField>
<ResponseField name="status" type="string">订单状态：`open`、`filled`、`partially_filled`、`cancelled`
<ResponseField name="postOnly" type="boolean">是否为 Post-Only 模式</ResponseField>
<ResponseField name="expiration" type="number">到期时间（Unix 秒），仅 GTD 订单有此字段</ResponseField>
<ResponseField name="createdAt" type="string">创建时间（ISO 8601）</ResponseField>
<ResponseField name="updatedAt" type="string">最近更新时间（ISO 8601）</ResponseField>

## 请求示例

```bash
curl "https://senalback.xyz/v1/mm/orders/order_01HXYZ001" \
  -H "SENAL-API-KEY: your_api_key" \
  -H "SENAL-SIGNATURE: your_signature" \
  -H "SENAL-TIMESTAMP: 1735689600000" \
  -H "SENAL-NONCE: random_nonce_16bytes" \
  -H "SENAL-PASSPHRASE: your_passphrase"
```

## 响应示例

```json
{
  "id": "order_01HXYZ001",
  "marketId": "market_001",
  "tokenId": "0xabc123def456",
  "side": "BUY",
  "orderType": "GTC",
  "price": "0.6400",
  "size": "200",
  "filledSize": "50",
  "remainingSize": "150",
  "status": "partially_filled",
  "postOnly": false,
  "createdAt": "2025-01-01T00:00:00.000Z",
  "updatedAt": "2025-01-01T00:05:00.000Z"
}
```

订单不存在时返回 HTTP 404：

```json
{
  "statusCode": 404,
  "message": "Order not found"
}
```
