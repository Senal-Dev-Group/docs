## 连接地址

```
wss://api.senal.market/v1/mm/ws
```

每个 API Key 最多允许 **5 个并发连接**。

## 认证

建立 WebSocket 连接后，**第一个消息必须是认证包**，否则连接会在短时间内被服务端关闭。

### 认证消息格式

```json
{
  "event": "auth",
  "data": {
    "apiKey": "your_api_key",
    "signature": "hmac_signature_hex",
    "timestamp": "1735689600000",
    "nonce": "random_nonce_string",
    "passphrase": "your_passphrase"
  }
}
```

### WS 签名算法

WebSocket 使用独立的签名规则（与 HTTP L2 类似，但 Method 固定为 `"WS"`，path 固定为 `"/v1/mm/ws"`）：

```
message   = timestamp + nonce + "WS" + "/v1/mm/ws"
signature = HMAC-SHA256(apiSecret, message)
```

```typescript
import { createHmac } from 'crypto';

function signWsAuth(apiSecret: string, timestamp: string, nonce: string): string {
  const message = timestamp + nonce + 'WS' + '/v1/mm/ws';
  return createHmac('sha256', apiSecret).update(message).digest('hex');
}

// 使用
const timestamp = Date.now().toString();
const nonce = crypto.randomUUID();
const signature = signWsAuth(apiSecret, timestamp, nonce);

ws.send(JSON.stringify({
  event: 'auth',
  data: { apiKey, signature, timestamp, nonce, passphrase },
}));
```

### 认证响应

成功：
```json
{
  "event": "auth",
  "data": { "status": "ok" }
}
```

失败：
```json
{
  "event": "error",
  "data": { "code": 401, "message": "Invalid signature" }
}
```

## 订阅频道

认证成功后，发送订阅消息：

```json
{
  "event": "subscribe",
  "data": {
    "channels": ["orderbook:0xabc123", "orders"]
  }
}
```

取消订阅：

```json
{
  "event": "unsubscribe",
  "data": {
    "channels": ["orderbook:0xabc123"]
  }
}
```

## 可订阅频道

### `orderbook:<tokenId>`

实时订单簿快照，每次有订单变化时推送。

```json
{
  "event": "update",
  "channel": "orderbook:0xabc123",
  "data": {
    "tokenId": "0xabc123",
    "bids": [
      { "price": "0.6500", "size": "500" },
      { "price": "0.6400", "size": "1000" }
    ],
    "asks": [
      { "price": "0.6600", "size": "300" },
      { "price": "0.6700", "size": "800" }
    ],
    "timestamp": "1735689600000"
  }
}
```

### `trade:<tokenId>`

指定市场的实时成交记录。

```json
{
  "event": "update",
  "channel": "trade:0xabc123",
  "data": {
    "id": "trade_001",
    "tokenId": "0xabc123",
    "price": "0.6550",
    "size": "100",
    "side": "BUY",
    "timestamp": "1735689600000"
  }
}
```

### `orders`

**私有频道**。推送当前 API Key 关联用户的订单状态变化。

```json
{
  "event": "update",
  "channel": "orders",
  "data": {
    "id": "order_001",
    "status": "PARTIALLY_FILLED",
    "filledSize": "50",
    "remainingSize": "50",
    "updatedAt": "1735689600000"
  }
}
```

### `positions`

**私有频道**。推送仓位变化。

```json
{
  "event": "update",
  "channel": "positions",
  "data": {
    "tokenId": "0xabc123",
    "size": "150",
    "avgPrice": "0.6520",
    "updatedAt": "1735689600000"
  }
}
```

## 频道汇总

| 频道 | 类型 | 说明 |
|------|------|------|
| `orderbook:<tokenId>` | 公开 | 指定 token 的订单簿变化 |
| `trade:<tokenId>` | 公开 | 指定 token 的成交记录 |
| `orders` | 私有 | 当前用户的订单状态变化 |
| `positions` | 私有 | 当前用户的仓位变化 |

## 连接维护

服务端会定期发送 Ping 帧，客户端需回复 Pong 帧（大多数 WebSocket 库自动处理）。

如果连接意外断开，应实现指数退避重连并在重连后重新认证和订阅频道。

```typescript
function connectWithRetry(delay = 1000) {
  const ws = new WebSocket('wss://api.senal.market/v1/mm/ws');

  ws.on('open', () => {
    // 1. 发送认证
    // 2. 重新订阅之前的频道
  });

  ws.on('close', () => {
    const nextDelay = Math.min(delay * 2, 30_000); // 最大 30 秒
    setTimeout(() => connectWithRetry(nextDelay), delay);
  });
}
```

<Note>
  WebSocket 连接断开不会触发心跳超时。心跳和 WebSocket 是独立机制，需要分别维护。
</Note>
