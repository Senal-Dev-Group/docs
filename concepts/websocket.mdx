Senal 做市商 API 提供 WebSocket 接口，用于实时接收订单簿、成交、订单状态和持仓变动推送。

<Note>
  详细的连接认证、订阅格式和消息 schema 请参考 [WebSocket API 参考](/api-reference/websocket/overview)。
</Note>

## 连接地址

```
wss://senalback.xyz/v1/mm/ws
```

每个 API Key 最多允许 **5 个并发连接**。

## 可用频道

| 频道 | 类型 | 说明 |
|------|------|------|
| `orderbook:<tokenId>` | 公开 | 指定 token 的订单簿变化 |
| `trade:<tokenId>` | 公开 | 指定 token 的成交记录 |
| `orders` | 私有 | 当前用户的订单状态变化 |
| `positions` | 私有 | 当前用户的仓位变化 |
